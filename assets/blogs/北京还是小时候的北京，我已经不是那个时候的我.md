# 【叁贰陆】这一次又拍到了故宫的冬天！

&emsp;这次去北京想起来很多童年的故事，再次见到故人，仿佛记忆就在昨天，虽然此次去是以真实姓名与大家相处，但每次听到有人喊我老二或者慕水的时候都会有一种非常强的亲切感。

&emsp;带着史铁生的《我与地坛》来到地坛，想象着史铁生描述的场景，那个寂静幽长的人行道，坐在轮椅上浮想联翩，一坐就是一下午，到了晚饭时间他的母亲推他回家～
![](https://files.mdnice.com/user/25819/65016f58-18e5-4bfb-bc11-7b06a28607cb.png)

![](https://files.mdnice.com/user/25819/f1671bff-28d8-40c9-9f71-9e3d99bf39bb.png)

&emsp;除了地坛让我印象深刻，就是北京的故宫了，大学有一次来，带着从老师那里借来的相机一个人到处扫街，到故宫时刚好是下雪，才发现北京故宫的魅力，那次之后一直都想再次遇到那样的场景，五年后，上天眷顾！
