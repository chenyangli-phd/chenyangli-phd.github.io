# 用代码模拟男女生吵架｜把爷整乐了

你以为的样子：
  ```
GF.start_fighting()
if BF.start_fighting();then
    GF.cry()
    GF.break_up()
elif BF.keep_silence();then
    GF.show_mercy()
```
然而现实可能是
```

GF.start_fighting()
while BF.start_fighting();do
    GF.cry()
    GF.anger()
    GF.curse()
    GF.philosophic()
    GF.unpredictable()
done
while BF.keep_speaking();do
    GF.argue()
    GF.yell()
    GF.cry()
    GF.unpredictable()
done
while BF.keep_silence(); do
    GF.yell()
    GF.cry()
    GF.EMO()
    GF.unpredictable()
done

while ! GF.show_mercy(); do
    if BF.break_up() || GF.break_up();then
        BF.release()
        GF.cry()
        GF.curse()
        GF.philosophic()
        GF.sleep()
        BF = GF.get_new_one()
        break;
    elif BF.kneel_down(); then
        GF.EMO()
        continue
    elif GF.apologize(); then
        BF.abort()
        GF.release()
        break
    fi
done
GF.dump()
  ```
